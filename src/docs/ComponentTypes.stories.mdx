<Heading size="xl" id="json-form" caption="JSON Format">Component types</Heading>

The following components are able to be configured within the JSON.
Also see <Link href="/?path=/docs/f-editable-components">editable components</Link>.

## `autocomplete`
Search for and select an item from a (usually large) set of options.
See <Link href="/?path=/docs/f-examples-component--page#autocomplete">example</Link>.

## `email`
Enter an email address.
See <Link href="/?path=/docs/f-examples-component--page#email">example</Link>.

## `file`
Select a file for upload.
See <Link href="/?path=/docs/f-examples-component--page#file">example</Link>.

## `heading`
A `govuk-heading` heading component (`h1`, `h2`, etc.).
See <Link href="/?path=/docs/f-examples-component--page#heading">example</Link>.

## `html`
A simple rendering of HTML, such as a paragraph, image, or horizontal rule.
See <Link href="/?path=/docs/f-examples-component--page#html">example</Link>.

## `inset-text`
Use the inset text component to differentiate a block of text from the content that surrounds it, for example:

* quotes
* examples
* additional information about the page

See <Link href="/?path=/docs/f-examples-component--page#inset-text">example</Link>.

## `phone-number`
Enter a telephone number.
See <Link href="/?path=/docs/f-examples-component--page#phone-number">example</Link>.

## `radios`
Select a single option from a (relatively small) set of options.
See <Link href="/?path=/docs/f-examples-component--page#radios">example</Link>.

## `text`
Enter a single line of free text.
See <Link href="/?path=/docs/f-examples-component--page#text">example</Link>.

## `textarea`
Enter multiple lines of free text.
See <Link href="/?path=/docs/f-examples-component--page#textarea">example</Link>.

## `date`
Use the date component when you want users to input a date.
See <Link href="/?path=/docs/f-examples-component--page#date">example</Link>.

## `time`
Use the time component when you want users to input a time.
See <Link href="/?path=/docs/f-examples-component--page#time">example</Link>.
