<Heading size="xl" id="json-format" caption="JSON Format">Editable components</Heading>

The following components are editable.

## `autocomplete`
Search for and select an item from a (usually large) set of options.

## `email`
Enter an email address.

## `file`
Select a file for upload.

## `phone-number`
Enter a telephone number.

## `radios`
Select a single option from a (relatively small) set of options.

## `text`
Enter a single line of free text.
